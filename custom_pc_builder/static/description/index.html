Custom PC Builder Module

About

This is an advanced Odoo version 16 CE module for custom PC building on an e-commerce website. It provides full fledged features and is a complete industrial module. It is designed from scratch to production level.

Features

  • Custom PC configuration
  • Product management
  • Sale order management
  • Stock management
  • Partner management

Installation

Place the 'custom_pc_builder' folder inside your Odoo addons directory and update the app list in your Odoo instance. Install the module from the app list.
